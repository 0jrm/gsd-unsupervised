Implement in `src/config/clipIntegration.ts`:
- Export a function `getClipCommand(): string | null` that returns the shell command to send stdin to the Windows clipboard:

- Keep the script intentionally conservative: it should primarily read from `which`/PATH and set env vars only when they are currently empty, so it does not override user configuration.
