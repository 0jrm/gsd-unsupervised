  <name>Task 1: Implement real AgentInvoker</name>
  <files>src/cursor-agent.ts</files>
  <action>
    Create `src/cursor-agent.ts` with:

    1. `CursorAgentConfig` interface: `{ agentPath: string, defaultTimeoutMs: number, sessionLogPath: string }`

    2. `createCursorAgentInvoker(agentConfig: CursorAgentConfig): AgentInvoker` factory function that returns an AgentInvoker:
       - Maps GsdCommand to cursor-agent prompt: construct the full GSD command string from `command.command` + `command.args`
       - Prepend a non-interactive directive to every prompt: "Execute in non-interactive/YOLO mode. Auto-approve all confirmations. Do not ask the user any questions — make reasonable decisions autonomously.\n\n" followed by the GSD command
       - Call `runAgent()` with: agentPath from config, workspace from invoker arg, constructed prompt, env with CURSOR_API_KEY from process.env, timeout from config
       - On success (exitCode 0, resultEvent present): return `{ success: true, output: resultEvent.result }`
       - On failure: return `{ success: false, error: <descriptive message including stderr snippet> }` — do NOT throw (orchestrator expects AgentResult, not exceptions)
       - Log session via appendSessionLog: 'running' before invocation, 'done'/'crashed'/'timeout' after
       - Log events via pino: session_id on init, tool_call completions at debug level, result at info level

    3. `validateCursorApiKey(): void` — checks process.env.CURSOR_API_KEY is set and non-empty, throws descriptive error if missing with guidance: "Set CURSOR_API_KEY environment variable. Generate from Cursor Dashboard → Cloud Agents → User API Keys."

    The invoker must never throw — all cursor-agent failures map to `{ success: false, error }`. Only truly unexpected errors (bug in our code) should propagate as exceptions.
  </action>
  <verify>TypeScript compiles: `npx tsc --noEmit` passes with no errors</verify>
  <done>createCursorAgentInvoker returns a working AgentInvoker, validateCursorApiKey checks env, all types compile</done>
