<task type="auto">
  <name>Task 1: Build orchestrator with stub agent</name>
  <files>src/orchestrator.ts</files>
  <action>
Create src/orchestrator.ts with:

1. **AgentResult interface:**
   ```
   {
     success: boolean;
     output?: string;
     error?: string;
   }
   ```

2. **Export type AgentInvoker = (command: GsdCommand, workspaceDir: string, logger: Logger) => Promise<AgentResult>**
   — This is the seam Phase 3 replaces. A function that takes a GSD command and returns a result.

3. **stubAgent: AgentInvoker** (not exported — internal default)
   - Log at info level: `Stub: would invoke cursor-agent with "${command.command} ${command.args ?? ''}" in ${workspaceDir}`
   - Return { success: true, output: 'stub' }
   - Add a 100ms delay (setTimeout wrapped in Promise) to simulate work and make logs readable

4. **Export async function orchestrateGoal(options: { goal: Goal, config: AutopilotConfig, logger: Logger, agent?: AgentInvoker, isShuttingDown: () => boolean }): Promise<void>**

   This is the main loop:

   a. Create GoalStateMachine for the goal
   b. Create child logger: createChildLogger(logger, 'orchestrator')
   c. Use agent = options.agent ?? stubAgent

   d. **Phase: new → initializing_project**
      - Check isShuttingDown() before each step
      - Get next command from state machine (should be /gsd/new-project)
      - Log command: "Executing: /gsd/new-project"
      - Call agent(command, config.workspaceRoot, logger)
      - If failed, call sm.fail(result.error), return
      - sm.advance(GoalLifecyclePhase.initializing_project)

   e. **Phase: initializing_project → creating_roadmap**
      - Get next command (/gsd/create-roadmap)
      - Call agent, advance on success

   f. **Phase: creating_roadmap → loop through phases**
      - Parse roadmap: parseRoadmap(path.join(config.workspaceRoot, '.planning/ROADMAP.md'))
      - sm.setPhaseInfo(1, phases.length)
      - Log: "Roadmap has {N} phases"

   g. **Phase loop: for currentPhase = 1 to totalPhases:**
      - Check isShuttingDown()
      - sm.advance(GoalLifecyclePhase.planning_phase)
      - Get command (/gsd/plan-phase N), call agent

      - Discover plans: findPhaseDir + discoverPlans
      - sm.setPlanInfo(1, plans.length)
      - Log: "Phase {N} has {M} plans"

      - **Plan loop:** while nextPlan = getNextUnexecutedPlan:
        - Check isShuttingDown()
        - sm.advance(GoalLifecyclePhase.executing_plan)
        - Build execute command: { command: '/gsd/execute-plan', args: nextPlan.planPath, description: `Execute plan ${nextPlan.planNumber}` }
        - Call agent
        - sm.setPlanInfo(nextPlan.planNumber + 1, plans.length)
        - Re-discover plans (summary may now exist)

      - sm.advance(GoalLifecyclePhase.phase_complete)
      - Log: "Phase {N} complete"
      - sm.setPhaseInfo(currentPhase + 1, totalPhases)

   h. **All phases done:**
      - sm.advance(GoalLifecyclePhase.complete)
      - Log: "Goal complete: {title}"

   Handle errors: wrap the entire loop in try/catch. On error, sm.fail(error.message), log error, re-throw.

   When isShuttingDown() returns true: log "Shutdown requested — stopping orchestration after current step at phase {phase}, plan {plan}" and return early (don't throw — graceful exit).
