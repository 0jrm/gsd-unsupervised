Create src/roadmap-parser.ts with:

1. **PhaseInfo interface:**
   ```
   {
     number: number;          // 1, 2, 3 or 2.1, 2.2 for decimals
     name: string;            // "Foundation & CLI Scaffold"
     description: string;     // "Node.js project setup, CLI entry point, ..."
     dirName: string;         // "01-foundation-cli-scaffold"
     complete: boolean;       // [x] vs [ ]
   }
   ```

2. **Export async function parseRoadmap(roadmapPath: string): Promise<PhaseInfo[]>**
   - Read the file (throw descriptive error if not found)
   - Parse each line matching the pattern: `- [x] **Phase N: Name** — Description` or `- [ ] **Phase N: Name** — Description`
   - Regex: `/^- \[([ xX])\] \*\*Phase (\d+(?:\.\d+)?): (.+?)\*\* — (.+)$/`
   - Extract: complete (x/X = true), number (parse as float), name, description
   - Return array sorted by phase number

3. **Export function findPhaseDir(phasesRoot: string, phaseNumber: number): string | null**
   - List directories in phasesRoot
   - Match directory starting with zero-padded phase number (01-, 02-, 02.1-)
   - Handle both integer (01-name) and decimal (02.1-name) formats
   - Return full path to matching directory, or null if not found

Use readFile from fs/promises (consistent with goals.ts pattern). Handle edge cases: empty file returns empty array, missing file throws with path in message, lines not matching pattern are silently skipped.

<task type="auto">
  <name>Task 2: Plan discovery within phase directories</name>
  <files>src/roadmap-parser.ts</files>
  <action>
Add to src/roadmap-parser.ts:

1. **PlanInfo interface:**
   ```
   {
     phaseNumber: number;     // 1, 2, etc.
     planNumber: number;      // 1, 2, 3 within phase
     planPath: string;        // full path to PLAN.md
     summaryPath: string;     // expected path to SUMMARY.md
     executed: boolean;       // true if SUMMARY.md exists
   }
   ```

2. **Export async function discoverPlans(phaseDir: string): Promise<PlanInfo[]>**
   - List all files in phaseDir matching `*-PLAN.md` pattern
   - For each PLAN.md, extract phase number and plan number from filename
   - Filename pattern: `NN-MM-PLAN.md` where NN is phase, MM is plan (both can be decimal)
   - Regex: `/^(\d+(?:\.\d+)?)-(\d+)-PLAN\.md$/`
   - Check if corresponding `NN-MM-SUMMARY.md` exists (use existsSync for simplicity)
   - Return array sorted by plan number

3. **Export function getNextUnexecutedPlan(plans: PlanInfo[]): PlanInfo | null**
   - Find the first plan where executed === false
   - Return it, or null if all plans are executed

4. **Export function isPhaseComplete(plans: PlanInfo[]): boolean**
   - Return true if plans.length > 0 AND all plans have executed === true
   - Return false if no plans exist (phase not yet planned)

Use readdir from fs/promises for directory listing. Handle edge cases: empty directory returns empty array, directory not found throws with path in message.
  </action>
